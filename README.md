# Norkart Kontorkart Demo – React + Leaflet + TypeScript

Full-stack demo laget på én uke fra scratch spesifikt for å vise interesse for Norkart.

### Funksjoner

- Interaktiv Leaflet-kart med alle Norkart-kontorer
- CRUD-operasjoner (legg til, slett, reset)
- Kristiansand mangler med vilje – prøv å legge det til!
- Optimerte oppdateringer med React Query
- Responsivt design og norsk UI

Backend (.NET 10): https://github.com/Zmax-17/norkart-backend-demo

Takk for oppmerksomheten – gleder meg til å bidra til smartere samfunn hos dere!
